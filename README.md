# AmalgamaExerciseArmies

Este repositorio contiene la resolución del ejercicio **Exercise Armies**, implementado utilizando **Smalltalk en el entorno Pharo**.

## Entorno y lenguaje
- **Lenguaje**: Smalltalk  
- **IDE / Entorno**: Pharo

El desarrollo fue realizado utilizando una **imagen de Pharo 13**.  
Sin embargo, el código **no depende de características específicas de esa versión**, por lo que **es compatible y funciona correctamente en versiones anteriores de Pharo** (por ejemplo Pharo 11 o 12).

## Cómo cargar el proyecto en Pharo
1. Abrir Pharo (versión 11 o superior recomendada).
2. Ir a **Sources → Git Repositories Browser → Add **.
3. Clonar este repositorio desde GitHub.
4. Cargar los packages:
   - `ExerciseArmies`
   - `ExerciseArmiesTest`
